BiliDownloader - B站视频下载与音频剪辑工具 📺🎵
一个简洁、现代的 Android 工具应用，集成了 Bilibili 视频下载与本地音频处理功能。
旨在提供流畅的 Material Design 3 使用体验，支持动态主题取色与暗色模式。
⚠️ 声明：本项目仅供技术研究和个人学习使用。请勿用于下载版权受保护的内容，严禁用于商业用途。
✨ 核心功能
1. 📥 B站视频解析与下载
   全能解析：支持输入 BV 号或 b23.tv 短链接进行解析。
   高清画质：支持下载 1080P 高清视频 (AVC/HEVC) 及高音质音轨。
   自动处理：
   合并：自动将分离的视频流和音频流合并为标准的 MP4 文件。
   提取：支持仅下载音频并自动转码为 MP3 格式。
   稳定可靠：内置 WBI 签名算法，有效解决接口鉴权问题。
2. 🎵 音频工作室 (本地文件管理)
   全盘扫描：快速扫描并展示手机内的音频文件，支持多种排序方式（时间、大小、时长）。
   文件管理：
   搜索：顶部实时搜索框，快速定位目标音频。
   操作：长按列表支持 重命名、删除 文件（适配 Android 10+ 分区存储权限）。
   分享：支持一键调用系统分享面板，将音频发送至 QQ、微信或其他应用。
3. ✂️ 专业音频裁剪
   可视化操作：提供直观的双向滑块，轻松选择裁剪区间。
   高精度控制：
   支持 毫秒级 (00:00.000) 手动输入起始与结束时间。
   输入框与滑块实时双向同步，精准定位。
   实时预览：
   支持裁剪片段的循环试听。
   包含实时播放进度条与时间显示。
   无损导出：裁剪完成后自动导出并保存至系统音乐库。
4. 🎨 现代 UI 设计
   Material Design 3：遵循最新的 Android 设计规范，界面美观大方。
   动态取色 (Monet)：支持 Android 12+ 动态颜色，APP 主题色随手机壁纸自动变化。
   暗色模式：完美适配系统深色主题，夜间使用更护眼。
   历史记录：仿原生风格的历史记录列表，支持长按批量管理。
